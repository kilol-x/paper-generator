# 论文生成系统运行说明

完整项目已统一保存在本文件夹中。

## 前端（Vue 3）

```bash
cd frontend
npm install
npm run dev
```

访问：http://localhost:5173

主要源码：

- `frontend/src/App.vue`：管理员、学生、教师统一入口
- `frontend/src/components/AdminSystem.vue`：合并后的管理员系统
- `frontend/src/style.css`、`frontend/src/system.css`：页面样式
- `frontend/vite.config.js`：Vite 及接口代理配置

## 后端（Spring Boot）

正式 MySQL 配置位于 `src/main/resources/application.yml`。

本地演示可以运行：

```bash
mvn "-Dspring-boot.run.profiles=demo" spring-boot:run
```

后端地址：http://localhost:8080
